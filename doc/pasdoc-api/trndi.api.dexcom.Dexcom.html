Description

****************************************************************************** Dexcom class

Inherits from TrndiAPI and implements connectivity and reading retrieval for Dexcom Share. Responsible for session handling, time-sync, and mapping Share responses into BGResults. *****************************************************************************

function Connect: boolean; override;

** Authenticate with Dexcom Share, establish a session, and synchronize time.

Workflow: 1) Authenticate to get initial session token. 2) Validate session. 3) Call login-by-name to finalize session. 4) Retrieve server UTC time and compute timeDiff.

Returns

True if session established and time synchronized; otherwise False and errormsg is set.
