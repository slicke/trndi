type xDrip = class(NightScout)

Description

** xDrip client. Inherits from NightScout and adapts certain behaviors for xDrip-based endpoints. Provides methods to: - Connect to the xDrip server - Retrieve glucose readings - Synchronize local time offset - Fetch BG threshold values (hi/lo)

constructor Create(user, pass, extra: string); override;

** Create an xDrip API client.

Initializes: - HTTP User-Agent string - baseUrl = trimmed user + '/' - API secret header value as 'api-secret=' + SHA1(pass) if provided - timezone and native HTTP helper

Note: - This constructor does not call inherited Create; if base-class initialization changes (e.g., default thresholds via initCGMCore), consider invoking it or duplicating that logic here.

function Connect: boolean; override;

** Connect to the xDrip server and initialize time offset and thresholds.

Steps: 1) Verify URL scheme (http/https) 2) Request XDRIP_STATUS ("pebble") and extract server now timestamp 3) Compute timeDiff relative to local time 4) Request XDRIP_RANGES and parse bgHigh/bgLow thresholds (fallbacks if absent)

Notes: - This implementation parses values from raw JSON strings using simple substring operations; consider switching to a JSON parser for robustness.

Returns

True on success; otherwise False and errormsg is set
