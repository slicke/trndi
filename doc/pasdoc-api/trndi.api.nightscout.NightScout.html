constructor create(user, pass, extra: string); override;

** Create a NightScout API client. Initializes the HTTP User-Agent, derives the API base URL from user, prepares the API secret header from pass, and calls the inherited constructor.

Parameters
user
Base Nightscout URL, including scheme, e.g. https://example.com
pass
Nightscout API secret in plain text; hashed to SHA1 for header
extra
Reserved for future use
function connect: boolean; override;

** Connect to Nightscout, retrieve server status, thresholds, and set time offset. Requests NS_STATUS, validates/handles errors, parses serverTimeEpoch and settings.thresholds, and computes timeDiff relative to the local machine time.

Returns

True on success; False otherwise. On failure, errormsg is set.

function getReadings(minNum, maxNum: integer; extras: string; out res: string): BGResults; override;

** Fetch SGV readings and map them to BGResults. If extras is empty, uses NS_READINGS. Sends count=maxNum as a query parameter. Populates each reading with value, delta, trend, environment (device, rssi, noise), timestamp, and derived level.

Parameters
minNum
Unused here; time-span can be endpoint-dependent
maxNum
Maximum number of readings to fetch
extras
Optional endpoint path override, e.g. 'entries/sgv.json'
res
Out parameter receiving the raw JSON response string
Returns

Array of BGReading; empty on errors or unauthorized
