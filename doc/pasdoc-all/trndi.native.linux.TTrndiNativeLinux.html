class function isDarkMode: boolean; override;

** Desktop-aware dark mode detection. Order: 1) KDE Plasma via kreadconfig5: General/ColorScheme contains "Dark". 2) GNOME via gsettings: org.gnome.desktop.interface color-scheme (prefer-dark/default), then fallback to gtk-theme containing "-dark". 3) GTK_THEME environment variable contains "dark" (e.g. Adwaita:dark). 4) Fallback heuristic comparing clWindow vs clWindowText brightness.
